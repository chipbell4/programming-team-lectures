Lily's Birthday Party

This problem essentially asked you to enumerate every combination of $k$ children from a set of $n$ children.

With itertools in python, this is trivial: 

from itertools import combinations
kids = ['April', 'Andy', 'Chip']
for subset in combinations(kids, 2):
	for kid in subset
		print(kid)
The python website also hints at how they implement it

What if you don't have itertools?

Imagine a recursive formulation for generating the subsets. We will need to recursively call the combination builder on the remainder of the list (so we don't get duplicates) for each possible list tail. Our base case? An empty size has no printout. 

def print_subset(L, size, printout=''):
   if size is 0:
      print printout
      return

   for k in xrange(len(L)):
      print_subset(L[k+1:], size-1, printout + ' ' + L[k])
Windows

We want to simulate a window click, but need to determine which window was clicked. This problem is not terribly difficult, but good data structure use is critical.

Here are our data structures

  • A Window class that stores its own location, and can determine if a click occurred within it (via an $x$ and $y$ coordinate).
  • A stack of Window objects, ordered by who's on top, i.e. the topmost window is index 0, while the bottom is index $n-1$.

The algorithm
  • Create your window stack from the input parameters.
  • For each of the clicks
    • Loop over each window, seeing if the click occurs in that window.
    • If it does occur at some index $k$, report: window k+1 (since the windows are 1-indexed).
Spiral of Theodorus

We have this spiral generated by a special pattern. We need to extrapolate that pattern to higher numbers. Notice that the cases they provide are easy, but the input could be very large. This throws out some tricks like memoization and just looping. We're going to have to find a closed form!

Polar Coordinates

If we switch to polar coordinates, the $x,y$ becomes an $r,\theta$. The closed form for $r$ is easy: $r = \sqrt{n + 1}$. We could write $\theta$ as as a sum \[ \theta_n = \sum_{i=1}^n \arctan{\frac{1}{\sqrt{n}}} \]

A Weird (unobvious) Observation

If we actually code up this angle formula, we actually can find a pattern: \[ \theta_n = 2\sqrt{n} + C \] where $C = -2.157782996659 \ldots$

Why? The basic idea is if you expand the power series for $\arctan{\frac{1}{\sqrt{n}}}$ inside of the sum, you get $2\sqrt{k}$ plus a couple of other terms that are relatively small. A $n \rightarrow \infty$ these terms get so small they essentially become a constant. Here's a proof sketch

Pen Counts

This problem is asking us to calculate, given some integer $n$, the number of triangles with perimeter $n$ and integral side lengths. Another way of thinking of this is that we have an $n$ and we want to know how many solutions are there to the Diophantine Equations $n = a + b + c$, where $a \le b \le c$ and $a + b \gt c$.
